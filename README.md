# SmartResume AI

An intelligent AI Resume Builder and Job Matcher powered by **RAG (Retrieval-Augmented Generation)** and Groq LLM.

**Features:**
- Upload your resume (PDF) and paste a job description
- Get detailed match analysis and gap suggestions
- Generate a tailored resume optimized for the job
- Generate a professional cover letter
- Download the tailored resume as .txt

Built as an educational project demonstrating practical LLM integration and RAG pipelines.

Tech Stack: Gradio, Groq, Sentence-Transformers, FAISS, PyMuPDF
